JavaScript Math

Math.floor(Math.random()*100 + 1) returns a random integer between 1 and 100 (both included).

Math.sqrt function returns square root.

Math.round return rounded value

Math.pi returns value of Pi.

Math.abs returns absolute value

Math.pow returns power.

Math.floor returns the value of rounded down to its nearest integer.

Math.max returns max value from the function.

Math.min returns min value from the function.

Math.ceil returns rounded value up to its nearest Integer

Math.random returns random number between 0 and 1
